# Twitter bot

Simple twitter bot to tweet about NIX* configuration from reddit

## Install

git clone https://github.com/dedsxc/unixbot

python3 -m pip install -r requirements.txt

## Usage

In config.ini:
- Put your twitter API key
- Put your reddit API key
